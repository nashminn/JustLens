JustLens is a Flutter-based document scanning app targeting Android 7.0+ first, iOS later.

**Why:** User wants a clean, no-bloat scanner that exports as images or searchable PDFs with OCR.

**How to apply:** All decisions should favor simplicity and good UX. No ads for now. Plans live in `.claude/plan/`. Feature scope is deliberately limited for v1 — don't gold-plate.
